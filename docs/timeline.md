# 時間線與來源層級(研究紀錄)

> 把關鍵年代與本專案脈絡按時間排列,並對**每條標明來源層級**,方便查證與後續補強。
> 來源層級定義:**官方**(EA/Origin)>**百科**(Wikipedia)>**資料庫**(MobyGames)>
> **社群維基**(Ultima Codex)>**收藏者**(個人收藏整理)>**待考**(尚無可靠公開來源)。

## 系列與本作時間線

| 年代 | 事件 | 來源層級 | 來源 |
|---|---|---|---|

| 1982 | *Ultima II: The Revenge of the Enchantress*(反派 Minax) | 百科 | Wikipedia〈Ultima (series)〉 |
| **1983-03-03** | **Origin Systems** 由 Richard、Robert、Owen Garriott 創立 | 百科(內文) | Wikipedia〈Origin Systems〉內文(資訊框作 3/4,另列 Chuck Bueche) |
| **1983** | **《Ultima III: Exodus》** 於 **Apple II** 發行;為 Origin 第一款遊戲 | 百科 + 資料庫 | Wikipedia〈Ultima III: Exodus〉/〈Origin Systems〉、MobyGames |
| 1983-08-23 | EA 官方繁中產品頁標示之上市日期 | 官方(EA 平台標註) | EA 官方繁中〈創世紀3:魔胎〉 |
| 1983 前後 | C64、Atari 8-bit、IBM PC 等版本 | 百科 | Wikipedia〈Ultima III: Exodus〉 |
| 1984 | *Softline* 讀者票選史上第三受歡迎 Apple II 遊戲 | 百科(轉引) | Wikipedia〈Ultima III: Exodus〉 |
| 後續 | 移植至 Mac、Atari ST、Amiga(強化畫面) | 百科 + 資料庫 | Wikipedia、MobyGames |
| 後續 | 日本平台(FM-7/MSX/PC-88/PC-98/Sharp X1/FM Towns),多有日文化 | 社群維基 | Ultima Codex〈Translations of Ultima games〉 |
| 後續 | **NES/Famicom** 重製(美版 *Ultima: Exodus*) | 百科 + 資料庫 | Wikipedia、MobyGames |
| — | 原版之後另移植至 **13 個其他平台**(總述) | 百科 | Wikipedia〈Ultima III: Exodus〉 |
| 近代 | **LairWare** 的 Macintosh 版;原始碼由 **Leon McNeill** 以 **MIT** 釋出 | 一手(原始碼倉庫) | GitHub `beastie/ultima3` |
| 2026 | **本專案 u3-cht**:SDL2 移植 + 繁體中文化 + AppImage/Windows 釋出 | 一手(本 repo) | 本 repo、`docs/GAMEPLAY-STATUS.md` |
