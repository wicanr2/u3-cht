## 反派譜系:三部曲的因果

《Ultima III》的核心衝突源自前兩作反派的「遺產」。三者關係常被概括為:

```
Mondain (Ultima I)
   │  其徒弟 / 愛徒
Minax  (Ultima II)
   │  兩人的造物 / 後裔
Exodus (Ultima III)   ← 本作最終敵人
```

### Mondain(蒙登)
Ultima I 的大反派,黑暗時代的源頭。被擊敗後,其影響並未完全消散。

### Minax(米納克絲)
Ultima II 的大反派,常被描述為 Mondain 的徒弟/愛徒,為其復仇。亦被擊敗。

### Exodus(魔胎)— 本作最終敵人
- **身分**:Mondain 與 Minax 的「造物與後裔」(offspring and creation),為兩位創造者
  的毀滅尋仇。
- **本質(劇透)**:「既非人類、也非機器」,模稜兩可;劇情後段揭示其核心更像一台
  **電腦/人工智慧**。
- **弱點(劇透)**:須取得**四張魔法卡**,以**特定順序**插入其核心主機,引發爆炸方能終結。
- **據點**:**火焰之島(Isle of Fire)** 上的堡壘。

> 譯名說明:EA 官方繁中將副標 Exodus 譯為「**魔胎**」(《創世紀3:魔胎》);本移植專案
> 的視窗標題沿用早期常見譯法「**末日**」。兩者並存,詳見 [在地化筆記](localization-notes.md)。

## 正派:Lord British(不列顛王)

- 系列的核心人物、**本作時代 Sosaria(索薩利亞)的統治者**,也是 Richard Garriott 在
  遊戲中的化身。
  > 註:系列要到後期才把世界更名為 **不列顛尼亞(Britannia)**;在 Ultima III 的時空,
  > 世界仍是 Sosaria,本文不把 U3 的世界直接稱作 Britannia。
- 在本作中,**玩家受 Lord British 之召**前往討伐 Exodus。其城堡(LCB,Lord British's
  Castle)是玩家重要的補給與情報據點。

## 玩家:自建的四人隊伍(非固定主角)

與「單一固定主角」的作品不同,Ultima III **沒有預設姓名的主角**。玩家在開頭
**自行創立一支最多四人的隊伍**,逐一決定每位角色的:

- **種族(Race,共 5 種)**:人類 / 精靈 / 矮人 / 哈比族 / 毛絨族(Fuzzy)。
  各種族有不同的**屬性上限**(力量/敏捷/智力/智慧),例如精靈敏捷上限高、矮人力量上限高。
- **職業(Class,共 11 種)**:鬥士 / 牧師 / 巫師 / 竊賊 / 聖騎士 / 野蠻人 / 雲雀 /
  幻術師 / 德魯伊 / 鍊金術士 / 遊俠。職業決定可用武器/防具、能否施展牧師或巫師法術、
  以及偷竊/解除陷阱的能力。
- **性別與屬性點分配**。

職業大致可分為「純戰」「純施法」與「混合(half-X half-Y)」三類。例如:
- **鬥士**:可用任何武器與盔甲,但不能施法、不擅偷竊;
- **巫師/牧師**:分別依智力/智慧施展巫師/牧師法術,裝備受限;
- **遊俠**:近乎全才,武器/防具與雙系法術皆可,並略通偷竊與解除陷阱。

> 種族/職業的完整中文屬性與說明,遊戲內以「種族屬性上限」「職業說明」圖呈現
> (本專案已重繪為繁中,見 README 截圖與 [入門玩法導覽](gameplay-guide.md))。
