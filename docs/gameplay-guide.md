# 入門玩法導覽

> 原創中文整理的**新手導覽**,聚焦「如何開始」。不收錄原始手冊全文或受版權保護的
> 詳表/地圖;深入資料請參考官方手冊與社群維基(連結見文末)。畫面以本繁中化版為準。

## 整體流程一覽

```
片頭/標題 → 主選單 → 編組隊伍(建立角色→組隊) → 啟程冒險(進入世界)
        → 探索 Sosaria(城鎮/城堡/地城) → 取得四卡與弱點情報 → 火焰之島終結 Exodus
```

## 1. 主選單

標題畫面後進入主選單,四個按鈕:

- **編組隊伍**:建立/管理角色與隊伍(首次遊玩從這裡開始)。
- **調整選項**:音效 / 音樂 / 自動戰鬥 / 快速移動等開關。
- **啟程冒險**:帶**已成形的隊伍**進入世界地圖。
- **返回畫面**:回片頭/展示。

## 2. 創角:種族與職業

進「編組隊伍 → 建立角色」,依序決定**名字 → 職業 → 性別 → 屬性點**。創角畫面上方會顯示
**該種族的屬性上限**與**職業說明**(本專案已中文化)。

### 種族(屬性上限傾向)
| 種族 | 特點(相對傾向) |
|---|---|
| 人類 | 各屬性平均 |
| 精靈 | 敏捷上限高 |
| 矮人 | 力量上限高 |
| 哈比族 | 智慧上限高 |
| 毛絨族 | 智力/敏捷上限高、力量上限低 |

> 四項屬性:**力量(STR)/敏捷(DEX)/智力(INT)/智慧(WIS)**。智力影響巫師法術、
> 智慧影響牧師法術。

### 職業(共 11 種,擇要)
- **純戰**:鬥士(任何武器/防具,不能施法)、野蠻人(半鬥士半竊賊)。
- **施法**:巫師(依智力施巫師法術)、牧師(依智慧施牧師法術)。
- **混合**:聖騎士、雲雀、幻術師、德魯伊、鍊金術士。
- **全才**:遊俠(武器/防具佳,雙系法術皆可,略通偷竊/解陷阱)。

新手建議組合:**鬥士 + 牧師 + 巫師 + 遊俠**(前排肉、補血、輸出、全能)是穩健起手。

## 3. 組隊與啟程

建立角色後,於「編組隊伍」中**組成隊伍(Form)**,再回主選單選**啟程冒險**進入世界地圖。
畫面右側為**隊伍面板(sidebar)**:顯示每人**種族+職業短碼、生命/法力條、等級(級)、
食物(糧)**。

## 4. 在世界中行動(基本操作)

- **移動**:方向鍵在世界地圖上行走(會消耗食物與時間)。
- **進城鎮/城堡**:走到城鎮/城堡圖塊上以「**進入(E)**」進場,內有商店、NPC、補給。
- **對話**:對 NPC 使用「交談/Transact」並選擇方向與隊員。
- **戰鬥**:遇敵進入獨立的回合制戰鬥畫面,逐一指揮隊員攻擊/施法/移動。
- **地城**:第一人稱視角的迷宮探索,藏有寶物與關鍵的烙印標記。

> 食物歸零會傷害隊伍,記得在城鎮補給;善用旅店/神殿恢復狀態。

## 5. 初期目標(不劇透細節)

1. 練幾級、湊齊裝備與基本法術。
2. 取得**船隻**以跨越水域、擴大可探索範圍。
3. 蒐集情報:拜訪城鎮 NPC、地城與神殿,逐步逼近**四卡**與 **Exodus 弱點**(Time Lord)。
4. 完成準備後,前往**火焰之島**終結 Exodus(詳見 [世界觀](ultima-iii-overview.md))。

## 本繁中化版的操作小抄

- 主選單:`編組隊伍 / 調整選項 / 啟程冒險 / 返回畫面`。
- 隊伍面板:`精靈賊`(種族+職業)、`級 1`(等級)、`糧:149`(食物)、`不可施法`(無法力職業)。
- 底部提示:`等待`(過一回合)、`進入 城堡!`(進場)等。
